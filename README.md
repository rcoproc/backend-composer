Ruby on Rails
-------------

This application requires:

- Ruby 2.3.0
- Rails 5.0.1

- gem kaminari
- gem cocoon
- gem devise
- gem boostrap-saas
- gem haml
- gem puma
- gem pundit
- gem pry-meta
- gem haml-rails

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).
